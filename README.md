# NotMess

Простой и быстрый мессенджер. Авторизация через Telegram бот.

## Возможности
- 💬 Сообщения с файлами до 50MB
- 📞 Звонки через WebRTC
- 🎁 Подарки и звёзды
- 🔒 Авторизация через @notmess_autobot
- 🌙 Тёмная/светлая тема
- 🌍 Русский/английский язык
- ⭐ Избранное (долгое нажатие)

## Установка
1. Скопировать `engine/config.example.php` → `engine/config.php`
2. Заполнить данные MySQL
3. Импортировать `database.sql`
4. Включить `mod_rewrite` в Apache

## Требования
- PHP 7.4+
- MySQL 8.0
- Apache с mod_rewrite
